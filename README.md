# Resource Library

UI-only React + TypeScript + Vite homepage for a Resource Library.

## What this project is

- Single homepage only
- Built with shadcn-style UI components
- Upload/download PDF book experience as front-end UI only (no backend logic)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
